## 🎯 Paddle Integration Setup

### Étape 1: Créer un Compte Paddle
1. Va sur https://paddle.com/signup
2. Enregistre-toi avec ton email: `punkypanda97@gmail.com`
3. Remplis les informations:
   - Company name: TaskFlow Inc
   - Website: https://taskflow.app
   - Country: France

### Étape 2: Configurer les Plans
Dans le dashboard Paddle:
```
1. Crée 3 produits:
   - Free (Gratuit)
   - Pro ($9/mois)
   - Enterprise ($49/mois)

2. Configure les webhooks:
   URL: https://taskflow-api.railway.app/api/v1/paddle/paddle-webhook

3. Copie tes clés API
```

### Étape 3: Obtenir tes Clés
```env
PADDLE_API_KEY=your_key_here
PADDLE_VENDOR_ID=your_vendor_id
PADDLE_SECRET=your_secret
```
